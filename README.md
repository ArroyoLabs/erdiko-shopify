erdiko-shopify
==============

Shopify API Adapter

* Connect to the shopify API
* Use with or without Erdiko
* Convenient composer install

Version (SemVer)
----------------

0.1.1

Installation
------------

**with composer**
	require "erdiko/shopify": "0.1.*"
